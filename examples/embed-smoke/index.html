Cheatsheet

Import the ESM bundle after running npm run build:lib:

import { createIdeaEditor } from '../../dist/idea-editor.esm.js';

Full view (default)

App shell with top bar, editor area, and status bar. Container should fill available height.

createIdeaEditor({
  mode: 'both',
  container: '#full-host',
  config: { locale: 'en' },
});

Inline view

Compact textarea-like shell with rounded border. Set an explicit height on the container.

createIdeaEditor({
  mode: 'text',
  view: 'inline',
  container: '#inline-host',
  config: { statusBar: true },
});
Notes:
  • Bundle path is relative to this page: ../../dist/idea-editor.esm.js
  • Inline hosts need explicit height (e.g. height: 200px in CSS)
  • view defaults to 'full' when omitted

Config reference

Public SDK types from dist/index.d.ts. Optional fields marked with ?.

IdeaEditorCreateOptions

mode
Required. Editor capability: 'text', 'graphic', 'both', or 'read-only'.
container
Required. CSS selector string or HTMLElement to mount into.
view?
Presentation mode: 'full' (default) or 'inline'.
config?
Optional settings object — see IdeaEditorConfig below.

IdeaEditorConfig

document?
Initial document JSON. Defaults to a new empty document.
locale?
UI locale: 'en' or 'uk'.
i18nOverrides?
Partial translation dictionary to override built-in strings.
statusBar?
Inline view only. Show bottom status bar. Default false. Ignored in full view.
chrome?
Full-view chrome toggles — see IdeaEditorChromeConfig.
text?
Pass-through options for the text editor (readOnly, toolbars, blocks, etc.).
graphic?
Pass-through options for the graphic editor (readOnly, leftPanel, blocks, etc.).
shortcuts?
{ enabled?, entries? } — global shortcut manager config.
onReady?
(instance) => void — called after the editor is mounted.
onChange?
(doc) => void — called on document changes.
onModeChange?
(mode) => void — called when active mode switches between text and graphic.
extensions?
Reserved for future plugin extensions.

IdeaEditorChromeConfig (full view)

showTopBar?
Show document title bar with actions. Default true in full view.
showStatusBar?
Show character/block count footer. Default true in full view.
showModeSwitcher?
Show text/graphic mode toggle. Default true when mode: 'both'.
showImportExport?
Show JSON import/export buttons. Default true unless read-only.
showUndoRedo?
Show undo/redo buttons. Default true unless read-only.

IdeaEditorInstance (returned handle)

destroy()
Unmount editor and release listeners.
getDocument() / setDocument(doc)
Read or replace the current document.
getMode() / setMode(mode)
Read or set active mode ('text' | 'graphic').
getEventBus()
Access the internal event bus for custom integrations.
exportJSON() / importJSON()
Trigger JSON export/import dialogs.
getElement()
Root DOM element of the mounted editor shell.
